Água, Tecnologia e Futuro — Manifesto

Uma reflexão pública sobre usar recursos finitos (como água) para alimentar máquinas que diminuem o esforço humano — e alternativas éticas.

Manifesto

/* O conteúdo do manifesto foi inserido aqui. Edite à vontade. */ Este manifesto convida a uma reflexão clara e prática: temos legitimidade ética para gastar recursos finitos do planeta — especialmente água — para alimentar máquinas cuja finalidade é reduzir o esforço humano? Quando essa escolha preserva vidas, aumenta dignidade e promove sustentabilidade, pode ser justificada. Quando perpetua desperdício e destrói condições de vida futuras, é necessário reavaliá-la. Propomos três princípios: 1. Priorizar vida: recursos vitais (água, alimentos, ecossistemas) devem estar disponíveis para garantir necessidades humanas e ecológicas básicas. 2. Eficiência e circularidade: o uso de máquinas deve reduzir o consumo total de recursos (reciclagem de água, energia renovável, sistemas fechados), não aumentá-lo. 3. Justiça intergeracional: decisões tecnológicas precisam considerar as próximas gerações — não apenas ganhos imediatos. Práticas recomendadas: projetos que recuperem e recirculam água; tecnologias alimentadas por energia renovável; indicadores de impacto que considerem água virtual e pegada hídrica; participação comunitária nas decisões. Convidamos você a compartilhar, adaptar e transformar essas ideias em políticas, projetos e linguagem pública."

Como publicar no GitHub (rápido)

  1. Crie um repositório novo no GitHub (por exemplo: manifesto-agua).
  2. Envie este arquivo como index.html para a branch main.
  3. Vá em Settings → Pages → Deploy from branch → selecione main / / (root). Salve.
  4. Seu site ficará em https://SEU_USUARIO.github.io/manifesto-agua/ (aguarde alguns minutos).
